# ORAM Server (Render Deployment)

This folder contains the ORAM public API server used by the RoBoT CRT Terminal.

## Endpoints
POST /oram  
GET /health

## Deployment
1. Push this folder to GitHub (robot-oram-server)
2. Create new Web Service in Render
3. Select this repository
4. Add environment variable OPENAI_API_KEY
5. Deploy
